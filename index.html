Static images

Data was in an XLS file. Used the spreadsheet's inbuilt functions to aggregate the numbers and draw the charts.
Data was in a tab-delimited text file. Wrote a Python script to convert this file to a CSV one, and saved the CSV file as an XLS file. Then, used the spreadsheet's inbuilt functions to aggregate the numbers and draw the charts.
Data was in the form of JSON objects that the Twitter API sends back. Wrote a Python script to extract the tweet text, clean up the text for punctutaions, user names, and irrelevant words, and create the word cloud.

Interactive maps

Data was in Wikipedia articles. Created a CSV file of dates and place names, wrote a Python script to add the latitude and longitude values to these places, and then used the updated CSV file as an input on CartoDB.com to create a layered bubble map.

Data was in books. Created a CSV file of place names, used my Python script to add the latitude and longitude values to these places, and then used the updated CSV file as an input on CartoDB.com to create a torque map.
